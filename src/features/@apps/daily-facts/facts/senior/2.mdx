🧠 **Web Workers для тяжёлых задач**

Всё, что грузит основной поток, можно вынести в отдельный поток:

```jsx hideHeader
const worker = new Worker('worker.js');
worker.postMessage({ numbers: [1, 2, 3] });
worker.onmessage = e => console.log(e.data);
```

> Так интерфейс не будет «подвисать» при больших вычислениях.
